# dotvim

A configuration for [Vim](https://www.vim.org/), [Neovim](https://neovim.io), and [MacVim](http://macvim-dev.github.io/macvim/).

## Setup

```bash
git clone --recurse-submodules -j8 https://github.com/alphabetum/dotvim.git ~/.vim
```
